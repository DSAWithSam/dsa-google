### Recommended Order

Follow the order above. Each tutorial introduces or reinforces important patterns:
- **Hash Map usage** (Two Sum, Valid Anagram)
- **Stack basics** (Valid Parentheses)
- **Pointer manipulation** (Merge Two Sorted Lists)
- **Running minimum/greedy** (Best Time to Buy and Sell Stock)
- **Tree recursion** (Invert Binary Tree)
- **Binary Search template** (Binary Search)
- **Grid traversal (DFS)** (Flood Fill)

### How to Use These Tutorials

1. Read the problem statement and clarifying questions first
2. Try solving it yourself before looking at the solutions
3. Compare your approach with the brute force and optimal solutions
4. Understand the time/space tradeoffs
5. Try the common mistakes and follow-up questions

Mastering these 8 problems gives you a very strong foundation for Google interviews, especially the phone screening portion.
